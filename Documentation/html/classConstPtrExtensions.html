Finds the index of the value in the fiven span that starts on multiple of the value length

if the ptr would be 'abbcdd' and the value 'dd', the function would return 4, but if the value would be 'bb' the function would return -1 because 'bb' doesn't occur on index of multiple of the value.Length (2)
